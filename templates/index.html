Kindle-ready dashboard images

Use /render/<latitude>/<longitude> to fetch a clean PNG that fits your Kindle Paperwhite screen (default {{ width }}×{{ height }} pixels). Coordinates are now provided as path parameters instead of query parameters.

Battery overlay: add batteryLevel (0-100) and optional isCharging=true|false|yes|no to display device power status on the generated image.

Example: /render/{{ default_latitude }}/{{ default_longitude }}?batteryLevel=75&isCharging=yes

The rendered image now comes from a Typst widget layout with the Kindle-friendly styling shown in the demo.
