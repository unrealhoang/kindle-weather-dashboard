Weather widget template

The PNG renderer pulls current conditions from Open-Meteo and lays out temperature, humidity, and a short description. The HTML view uses this partial to illustrate how you can compose additional widgets or sections when expanding the dashboard.

  • Current temperature and “feels like” values
  • Humidity and Open-Meteo condition description
  • Optional battery status overlay for Kindle diagnostics
  • Override resolution with width and height query parameters if you use a different model
